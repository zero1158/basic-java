网络编程 : 在javaWeb中详细说明
    主要问题: 1.如何准确的定位一台或多台主机; 定位主机上的应用
             2.找到主机后如何进行高效可靠的数据传输
    解决方式: 1.IP和端口号
             2.提供网络通信协议,TCP/IP参考模型
                网络分层:
    DNS:域名解析服务器->将域名解析成IP地址,再去访问具体的网络服务器
    也可以在C:\Windows\System32\drivers\etc\hosts文件中指定具体对应的IP地址
    端口号的范围被限制在一个0-65535

    IP地址和端口号组合得出一个套接字 : socket

    网络通信协议:
        tcp:传输控制协议
        udp:用户数据报协议
        具体见图2,3
